# X (Twitter) Auto Liker

X (旧Twitter) で、特定の投稿以降を自動で「いいね」していくChrome拡張機能です。

## 🛡️ 主な機能と安全対策

このツールは「人間のような挙動」と「厳格な制限」を重視しています。

* **自動スクロール & 探索**: 最後にいいねした投稿を探し、それより下の投稿を順番にいいねします。
* **視覚的なターゲット表示**: 現在狙っているツイートを青枠で囲んで表示します。
* **ランダム待機**: いいねの間隔を **1秒〜3秒** の間でランダムに変動させ、機械的な動作をごまかします。
* **セッション制限 (49回)**:
    * 連続で **49回** いいねを行うと、自動的に動作を停止します。
    * その後、**15分間の強制クールダウン（ロック）** が発生し、解除されるまで再開できません。
* **1日の上限 (990回)**:
    * 1日の合計いいね数が **990回** に達すると、その日は動作を停止します。
    * 翌日（日付変更後）にカウントは自動リセットされます。

## 📂 ファイル構成

以下の2つのファイルを1つのフォルダ（例: `twitter-auto-liker`）に保存してください。

1.  `manifest.json` (設定ファイル)
2.  `content.js` (プログラム本体)

## 🚀 インストール方法

1.  PCで Google Chrome を開きます。
2.  アドレスバーに `chrome://extensions/` と入力して開きます。
3.  右上の **「デベロッパーモード」** をオンにします。
4.  左上の **「パッケージ化されていない拡張機能を読み込む」** をクリックします。
5.  作成したフォルダ（`twitter-auto-like`）を選択します。
6.  X (Twitter) のページを開き、再読み込み（リロード）します。

## 🎮 使い方

1.  Xのタイムライン、検索結果、リストなどを開きます。
2.  **「ここまで手動でいいねした」** というツイートが見える位置までスクロールしておくとスムーズです。
3.  画面右下に表示される黒いパネルの **「▶ 開始」** ボタンを押します。
4.  自動でターゲットを探し、いいねを開始します。

### パネルの見方
* **連続実行**: 現在のセッションで何回いいねしたか（49回で停止）。
* **本日合計**: 今日1日で何回いいねしたか（990回で停止）。
